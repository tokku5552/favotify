# favotify
Spotifyで今週一番再生した曲をTwitterに投稿するbotサービス
![](docs/archtecture.drawio.png)

## Environment
|tool|version|
|--|--|
|node|v18.12.1|
|npm|8.19.2|
|yarn|1.22.19|

# Getting Started

- リポジトリからクローンしてきたとき
```bash:
yarn install
```

- ローカルでのサーバ立ち上げ
```bash:
yarn dev
```
